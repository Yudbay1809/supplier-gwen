# Brand Logos

Simpan logo brand per SKU di folder ini.

## Format Disarankan
- PNG dengan background transparan
- Ukuran: 256x256 atau 512x512
- Nama file: lowercase-kebab-case, contoh:
  - gwen.png
  - lumiere.png
  - naturaglow.png

## Cara Pakai
Isi field `logoSrc` di `app/data/dashboard.ts`:

```ts
logoSrc: "/brand-logos/gwen.png"
```
