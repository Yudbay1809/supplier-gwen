# Contributing

Terima kasih sudah ingin berkontribusi.

## Alur singkat

1. Fork repo
2. Buat branch baru: `feature/nama-fitur`
3. Lakukan perubahan dan pastikan `npm run lint` lulus
4. Ajukan Pull Request

## Standar kode

- Gunakan TypeScript/React patterns yang konsisten
- Hindari inline style, pakai utility class
- Pastikan UI konsisten dengan warna brand
